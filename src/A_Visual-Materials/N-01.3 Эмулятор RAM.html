* emuStudio

* Извлекаем папку emuStudio с файлами программы из скачанного zip-файла в нужную директорию.
* Запуск emuStudio (способ 1):
  - В Командной строке переходим в директорию с файлами программы, где лежит файл emuStudio.bat,
  - В этой директории командой emuStudio.bat запускаем программу.
* Запуск emuStudio (способ 2):
  - В папке с файлами программы нажимаем правой кнопкой мыши на файл emuStudio.bat,
  - В контекстном меню нажимаем Создать ярлык,
  - В папке с файлами программы нажимаем правой кнопкой мыши на созданный файл emuStudio - ярлык,
  - В контекстном меню выбираем Отправить -> Рабочий стол (создать ярлык),
  - Кликнув по ярлыку на рабочем столе запускаем программу.
* После запуска в появившемся окне выбираем Random Access Machine(RAM) и нажимаем Open computer

* Таким образом можно посмотреть дополнительные сведения о компьютере Random Access Machine (RAM).

Инструкции (синим) алгоритма сложения чисел из входной ленты в RAM с пояснениями (черным).

* Работа с файлами в компьютере RAM:
1. В меню File выбираем Open,
2. Выбираем в директории нужный файл, кликаем Open,
3. Файл открывается в RAM. Файл должен быть с расширением .ram, в файле только инструкции (рис.13), каждая с новой строки, комментарии предваряются знаком точка с запятой.
При некорректном отражении кириллицы в комментариях, возможно надо поменять кодировку файла на UTF-8 c BOM.

* Работа с файлами в компьютере RAM (продолжение):
4. В меню Project жмем Compile,
5. Наш файл компилируется и в директории файла создается бинарный файл с тем же названием но с расширением .bram.

* Перейдя на вкладку Emulator можно двойным кликом открыть дополнительные окна Storage, Input Tape, Output Tape.
* В основном окне отражаются инструкции скомпилированного файла.
* Каждый клик по зеленой стрелке - шаг в алгоритме рабочего файла, вплоть до итогового результата.
* При этом результат выполнения инструкций на каждом шаге отражается в окне Storage, входная лента в Input Tape, итоговый результат в Output Tape.
* Для получения результата сразу - клик по синей стрелке.
