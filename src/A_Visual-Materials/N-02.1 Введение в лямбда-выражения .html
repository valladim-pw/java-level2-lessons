* В традиционном программировании мы оперируем с данными внутри цикла.
* Возможность использовать лямбда-выражения появилась начиная с JDK-8.
* При использовании лямбда-выражений применяется другой подход.
* В данном случае у нас есть набор данных (список) и мы в него с помощью метода (forEach) передаем поведение.
* Поведение передается в виде функции, которая работает внутри структуры данных и выдает необходимый результат (element -> System.out.println("element = " + element)).

* До лямбда-выражений уже был способ передачи поведения через анонимные классы как в примере выше.

* С появлением лямбда-выражений способ передачи поведения стал гораздо компактнее .
