<input> 1 2 3 4
READ 0 - читать первое число из входной ленты сумматором (регистр 0)
READ 1 - загрузить второе число из входной ленты в регистр 1
LOAD =0 - присвоить 0 в сумматор (регистр 0)
LOAD 2 - загрузить число из регистра 2 в сумматор (регистр 0)
STORE 3 - загрузить число из сумматора в регистр 3
ADD =3 - прибавить к сумматору число 3
ADD 3 - прибавить к сумматору число из регистра 3
SUB =2 - вычесть число 2 из сумматора
SUB 2 - вычесть из сумматора число из регистра 2
JMP mark - безусловный переход по метке mark
JGTZ mark - переход по метке mark, если число в сумматоре > 0
JZmark - перход по метке mark, если число в сумматоре = 0
WRITE 3 - число из регистра 3 записывается в выходную ленту
HALT - остановка выполнения программы
LOAD *2 означает косвенную ссылку
 Например, в регистре 2 лежит 10, 
 тогда LOAD *2 означает, 
 что в сумматор будет загружено содержимое регистра 10.
